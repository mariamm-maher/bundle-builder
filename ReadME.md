# Frontend Bundle Builder

This project is a React implementation of a multi-step bundle builder with a live review panel.

## Tech Stack

- React
- Tailwind CSS
- Context API
- Vite

## Installation

Clone the repository and navigate to the project directory:

```bash
cd project-folder
```

Install the dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

The application will then be available on the local development server provided by Vite.
